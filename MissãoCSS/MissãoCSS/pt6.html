SELETORES

Para que serve?

O CSS usa seletores para selecionar os elementos HTML corretos.

Quais são?

SELETOR FUNÇÃO SIMBOLO EXEMPLO
TAG Mira em tudas as tags com aquele nome p{ }
CLASSE Mira em tudo que possuir o atributo class . .destaque{ }
ID Mira em um elemento único que possui aquele ID # #menu_principal{ }
* Ele mira em TODOS os elementos da página. * *{ }
