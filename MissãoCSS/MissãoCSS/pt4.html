A “Receita” do CSS

O que é um Seletor?

O seletor é o ponto de partida de qualquer regra CSS. Ele funciona como um "endereçamento" que identifica quais elementos HTML receberão os estilos definidos.

O que é uma Propriedade?

A propriedade define qual aspecto visual do elemento será alterado. É a "característica" que queremos modificar.

O que é um Valor?

O valor define a configuração específica para a propriedade. Ele responde "como" a propriedade será aplicada.
