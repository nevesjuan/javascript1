Definições

O que é?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a aparência de documentos escritos em linguagens de marcação como HTML, controlando cores, fontes, layout, espaçamentos e outros aspectos visuais.

Qual a sua Função?

A principal função do CSS é estilizar e dar layout às páginas web escritas em HTML, ele seria igual a decoração de uma casa, onde você pode decorar o design conforme o limite da sua imaginação, porém, sem não houver ele, a casa (HTML) ainda vai ser funcional

Por que Separar o CSS do HTML?

Porque o código fica mais limpo e agradável, facilita nas modificações, onde para modificar é necessário apenas mudar um único arquivo para estilizar todas as páginas, além de ser necessário apenas mudar as informações no CSS para modificar algo no design. Também, ajuda na separação de tarefas, um exemplo são as empresas grandes, onde é separado quem irá fazer a estrutura (HTML) e o design (CSS).
